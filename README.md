# Getting Started

## Install

```
npm install @idealjs/layout-manager
```
or
```
yarn add @idealjs/layout-manager
```

## Example

- CodeSandbox
    > https://codesandbox.io/s/optimistic-bhabha-udcul

- Run locally

    ```sh
    git clone https://github.com/idealjs/layout-manager.git
    cd layout-manager
    yarn
    yarn build:webpack
    yarn link
    cd example
    yarn
    yarn link @idealjs/layout-manager
    yarn start
    ```
